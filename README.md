## Clone & install
```
git clone https://github.com/nocodb/nocodb-seed

cd nocodb-seed 

npm install
```

## Start
```
git clone https://github.com/nocodb/nocodb-seed

cd nocodb-seed 

npm install

npm run start
```

## To Upgrade
```
npm run upgrade
```

## Using environment file
Copy the .env.example file to .env and change as needed.
